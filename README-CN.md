# Thinkpad-E450-OpenCore
 HackInTosh - OpenCore 0.6.3
 
 MacOS version: 10.15.7 (19H15) - Catalina

## 正常使用：

- 亮度、显卡、网卡、声卡、等一切正常。
- 无线网卡需要配合[HeliPort](https://github.com/OpenIntelWireless/HeliPort/releases)使用。

## 问题:
- 电量百分比现实正常，电池充电状态插拔不切换状态。
